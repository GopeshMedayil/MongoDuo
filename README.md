# **MongoDuo**
A Desktop application developed using electronjs for importing mongodb schemas.
#### **Technologies used:**
- Electronjs
- Node.js
- Angular 7

#### **Install and Run**

```javascript
git clone https://github.com/GopeshMedayil/MongoDuo.git
cd MongoDuo
npm install
npm run build:dev

```
